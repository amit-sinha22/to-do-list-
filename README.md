# Simple Python To-Do App

This is a beginner-friendly command-line to-do app written in Python.

## Features

- View tasks
- Add tasks
- Remove tasks

## How to Run

Make sure Python 3 is installed, then run:

```bash
python3 todo_app.py
```

## File in This Project

- `todo_app.py` - the main Python program

## Notes

- Tasks are not saved after you close the app.
- This version is meant for learning basic Python concepts like lists, functions, loops, and input.
