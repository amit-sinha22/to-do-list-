# Simple Python To-Do App

This is a beginner-friendly to-do app project with:

- a command-line version in Python
- a browser version in HTML, CSS, and JavaScript

## Features

- View tasks
- Add tasks
- Remove tasks
- Mark tasks as complete
- Tasks are automatically saved to `tasks.json` in the Python version and `localStorage` in the browser version.

## How to Run

Make sure Python 3 is installed, then run:

```bash
python3 todo_app.py
```

## How to Run in a Browser

Open [index.html](/Users/amitkumar/Documents/New%20project/index.html) in any web browser.

## Files in This Project

- `todo_app.py` - the main Python program
- `index.html` - the browser-based version of the app

## Notes

- The Python version uses `tasks.json` for persistent storage and is meant for learning data structures, functions, loops, and I/O.
- The browser version uses `localStorage` for persistent storage and is meant for learning basic HTML, CSS, and JavaScript.
